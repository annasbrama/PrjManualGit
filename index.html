Comandos iniciais do Git

git --global user.name: criação do seu main/usuário.

git --global user.email: criação do email que se quer usar como main/usuário.

git init: primeiro comando dentro da pasta para o inicio do repositório.

git add "nome do arquivo": comando para a adição de arquivos novos ou modificados no repositório.

git commit -m "nome do status da criação ou modificação no arquivo": para deixar marcado no histórico do git dos seus passos a cada criação de arquivos ou modificação nos mesmos.

git remote add origin main "link do github": trazer para o git o link do repositório criado no github.

git push origin main: levar para o github todo o seu arquivo e suas modificações descritas através dos commits.

Comandos úteis e práticos

git status: o andamento e status do seu arquivo para se situar bem.

git remote -v: no caso de ter dúvida de qual usuário está conectado no git.

git pull: para trazer através do github um repositório e seus commits com atualizações.

git clone: para trazer através do github um repositório do zero em um local novo.
