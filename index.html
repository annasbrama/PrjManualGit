Instalação

Baixe o git para o OSX Baixe o git para o Windows Baixe o git para o Linux

Comandos iniciais Git

git --global user.name
criação do nome do main/usuário.
git --global user.email
criação do email do main/usuário.
git.init
dentro da pasta esse comando criará o repositório.

Comandos de criação Git

git.add"nome do arquivo"
comando para adição dos arquivos recém criados ou modificados dentro do repositório.
git add.
adiciona de uma só vez todos os arquivos ou modificações por um só comando.
git commit -m "comentários da alteração"
para deixar marcados todas as modificações feitas no seu arquivo através dos commits.
git remote add origin main "link do repositório criado no github"
trazer para o git o repositório também criado no github e mandar os arquivos para o hub.
git push origin main
leva para o github todas as modificações feitas no seu arquivo através dos seus commits.
